## 2. Cấu trúc trang web

Website được chia thành 6 phần chính:

### 2.1. Header (Thanh điều hướng)
- **Logo:** Hiển thị tên "Minh Đào" với thiết kế nổi bật
- **Menu điều hướng:** Gồm 5 mục
  - Trang chủ
  - Dịch vụ
  - Dự án
  - Về tôi
  - Liên hệ
- **Hamburger menu:** Hỗ trợ responsive trên thiết bị di động

### 2.2. Hero Section (Phần giới thiệu chính)
- Hiển thị lời chào "Xin chào, Tên tôi là Minh"
- Nút CTA (Call-to-Action) dẫn đến phần Portfolio
- Thiết kế ấn tượng, thu hút người xem ngay từ đầu

### 2.3. Services (Dịch vụ)
Giới thiệu 4 dịch vụ chính:

| Dịch vụ | Mô tả |
|---------|-------|
| **Cài win dạo** | Cài đặt Windows, cập nhật driver, tối ưu hệ thống |
| **Sửa máy tính** | Khắc phục lỗi phần cứng, phần mềm, virus, bảo trì định kỳ |
| **Thiết kế hệ thống** | Tư vấn và thiết kế hệ thống mạng, server cho doanh nghiệp |
| **Thiết kế web** | Thiết kế website chuyên nghiệp, responsive, tối ưu SEO |

### 2.4. Projects (Dự án)
Trình bày 5 dự án tiêu biểu:

1. **Website Bán Hàng Online**
   - Thương mại điện tử hoàn chỉnh
   - Tích hợp giỏ hàng, thanh toán trực tuyến

2. **Ứng Dụng Quản Lý Công Việc**
   - Quản lý task và dự án
   - Theo dõi tiến độ và báo cáo thống kê

3. **Landing Page Doanh Nghiệp**
   - Hiệu ứng animation đẹp mắt
   - Tối ưu tốc độ và SEO

4. **Hệ Thống Đặt Lịch Hẹn**
   - Đặt lịch trực tuyến cho phòng khám, salon, spa
   - Tích hợp nhắc nhở email/SMS

5. **Blog Cá Nhân**
   - Giao diện tối giản
   - Hỗ trợ markdown editor

## 4. Cấu trúc file

```
one-page-website-html-css-project/
├── index.html      # File HTML chính
├── style.css       # File CSS định dạng giao diện
├── app.js          # File JavaScript xử lý tương tác
├── img/            # Thư mục chứa hình ảnh
│   ├── img-1.png
│   └── img-2.png
├── README.md       # Hướng dẫn sử dụng
└── LICENSE         # Giấy phép
```

## 5. Hướng dẫn sử dụng

1. Mở file `index.html` bằng trình duyệt web
2. Hoặc sử dụng Live Server trong VS Code để xem trực tiếp
3. Có thể tùy chỉnh nội dung trong file `index.html`
4. Chỉnh sửa giao diện trong file `style.css`
